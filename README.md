# Project_TranscriptomicsCourse
Master's project on transcriptome - morphology association

In the framework of a course on functional genomics, we had to use a mock dataset relating morphological clustering and information on samples, together with clinical features of patients, and matched transcriptomics data. The idea of the project is to create a pipeline allowing to to correlate morphological variations with changes in gene expression. 

In the code published here, we first explored our samples and their clinical information. We then studied the association of clinical variables to morphological clusters and finally, the association of morphological clusters to transcriptomes and gene sets.
